Cloudiness


The purpose of the porject was to analyze how weather changes as you get closer to the equator. To accomplish this analyzes, we first pulled data from the open weather map api to assembe a data set on over 500 cities

After assembling the dataset, we used matplotlib to plot various aspects of the wqeather vs latitude. Factore we looked included; temprature cloudiness, windspeed and humidity. This site provides the source data and the visualization created as part of analysis, as well as explanation and description of any trens and corelation witnessed.
